# hyprPanopticon

## Concept

hyprland workspace overview. When launched, it displays workspaces that are open in Hyprland, arranged along a circumference. The focused workspace is set to maximum size, while the preview sizes of the other workspaces are automatically adjusted so that the entire layout remains visible.

## Versioning

Follow SemVer 2.0 to version each change appropriately and update the version in Cargo.toml.
